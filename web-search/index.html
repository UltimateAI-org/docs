Web Search

April 9, 2025

AI Models (Large Language Models) are trained, as such their knowledge will have a specific cut-off date.
TIP: Chat models are not self-aware and may return Knowledge Cut-Off dates.

If you would like to include the latest data \ information in your query there are two main approaches;

  • Select an (On-Line) model
  • Engauge (Web) mode, for an agent based approach

On-Line Models

On-line models provide rapid responses using cached but up-to-date information from the web.

Engauge Web Mode for Agentless Search

Web mode works with all Chat Models, and will use an agent based approach to add additional knowledge.  The web agent will use AI to produce a series of web searches, the agent will then visit the web pages, extracting the required information.
The agent based approach is slower, but more detailed.
