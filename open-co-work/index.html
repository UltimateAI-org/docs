Open Co-Work

May 7, 2026

Open-Co-Work, runs locally in an isolated sandbox.

How to Install

Download and install the latest release for Windows and Mac Open Co-Work Releases

OpenSource Repo  https://github.com/OpenCoworkAI/open-cowork/releases

Once Installed configure the API provider as follows;

  • Select "More Models"
  • Enter your API key from "settings->account"
  • Select "Anthropic" as the protocol
  • Base URL: https://chat.ultimateai.org
