App Installation

April 9, 2025

The Chat Application is available as a Progressive Web Application (PWA), for Windows, Android and iOS.

iOS App Installation


The installation of the PWA application is the same process as adding a website to your iOS home screen, except it will install as an Application and not a web-link,

To add a website to your iOS home screen, follow these steps:

  1. Open Safari: Navigate to https://chat.ultimateai.org using the Safari browser on your iPhone or iPad.

  2. Tap the Share Icon: Look for the share icon (a box with an arrow coming out of it) at the bottom of the screen and tap it.

  3. Scroll Down and Select "Add to Home Screen": In the share menu, scroll down until you see "Add to Home Screen" and tap it.

  4. Edit the Name (Optional): You can edit the name that will appear under the shortcut icon on your home screen.

  5. Tap Add: Once you're satisfied with the name, tap "Add" to add the shortcut to your home screen.

Windows App Installation

  1. Launch Edge and navigate to https://chat.ultimateai.org (NOT SAFE MODE)
  2. Look for the "App available" icon in the address bar (if the site supports PWA). Click it and select Install, or:
  3. Click the three-dot menu (Settings and More) in the top-right corner.
  4. Select Apps > Install this site as an app.
