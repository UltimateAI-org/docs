Coding

January 19, 2026

Coding via the API is permitted ONLY with Minimax-M2.1

You may use your own agents or IDE environments, however ensure they are configured to use the MiniMax-M2.1 model.

Install Claude Code

  1. Refer to the Claude Code documentation for installation.
  2. Edit or create the Claude Code configuration file located at %userprofile%\.claude\settings.json (Windows)
    ~/.claude/settings.json (Linux NOT WSL)

    Create or update as shown below, replace <your API key> with your API Key without <>;

{

  "env": {

    "ANTHROPIC_AUTH_TOKEN": "<your API key>",

    "ANTHROPIC_BASE_URL": https://chat.ultimateai.org,

    "API_TIMEOUT_MS": "3000000"

  },

  "model": "haiku"

}

Alternatively (including WSL users set the environment variables;

ANTHROPIC_BASE_URL="https://chat.ultimateai.org"

ANTHROPIC_AUTH_TOKEN="<your api key>"

API_TIMEOUT_MS="3000000"




Save the config and run claude.exe.

Claude Code Extension for VS Code (Optional)

  • Install Claude Code as shown earlier.
  • Install Claude Code Extension for VS Code

That's it.
