Flow Charts

Tim April 9, 2025

When requesting a Flow Chart, request a mermaid diagram to allow the LLM to display a graphic that you can download as an SVG file.

TIP: Top right of code blocks and graphics has action items such as "download, reset view and copy to clickboard"
